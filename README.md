# Stock Market Dashboard

Built this dashboard using Python and Streamlit to analyze stocks the way I personally trade — focusing on price action, trends, and key setups.

## What it does
- Pulls real market data for any ticker
- Shows price trends and charts
- Uses indicators like RSI and EMAs to break down momentum
- Helps identify potential trade setups quickly

## Tech Stack
- Python
- Streamlit
- Pandas
- yfinance

## How to run
```bash
pip install -r requirements.txt
streamlit run app.py
